# **App Name**: ideliver

## Core Features:

- User Profiles: User-friendly signup and profile creation for both customers and transporters.
- Request Submission: Submission form where customers can log and submit transport requests with destination details.
- Quote Submission: Transporters can respond to transport requests with their quotes.
- Credibility Verification: An AI powered tool will check transporter's profiles to decide whether or not the the user is safe and credible
- Quote Review: Customers can view and compare quotes from different transporters.
- Cargo Tracking: Real-time tracking of cargo movement from origin to delivery on a map.
- Proof of delivery: Visual confirmation of delivered packages

## Style Guidelines:

- Primary color: Strong blue (#29ABE2) to inspire confidence and security.
- Background color: Light gray (#F5F5F5), creating a neutral backdrop that focuses user attention.
- Accent color: Yellow-orange (#FDB827), highlighting important CTAs and sections.
- Headline font: 'Poppins', a geometric sans-serif, conveying a precise, contemporary, and fashionable feel, for titles.
- Body font: 'Inter', a grotesque-style sans-serif, to provide a neutral and modern readability.
- Consistent and easily recognizable icons for navigation and action items.
- Clean and intuitive layout for easy navigation and a streamlined user experience.
